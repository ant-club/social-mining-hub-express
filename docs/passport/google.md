# medium passport

## 开通

在[google开发者平台](https://console.developers.google.com/apis/credentials)申请OAuth 2.0 Client IDs即可。

## 配置

```json
{
  "PASSPORT": {
    "GOOGLE_CLIENT_ID": "",
    "GOOGLE_CLIENT_SECRET": "",
    "GOOGLE_CALLBACK_URL": "http://test.social-mining.com:3004/auth/google/callback"
  }
}
```
